Wat is Git?

Git is een versiebeheersysteem voor code die je schrijft om software/ websites te maken.

Maar, wat is een versiebeheersysteem?

Een versiebeheersysteem is een systeem die je gebruikt om verschillende versies van uw code op te slaan en navigeerbaar maken.
Dit wordt meestal gebruikt in combinatie met een website die uw repositories opslaat. Meer info in het volgende deel.

Weetje: git werd oorspronkelijk ontwikkeld door Linus Torvalds die ook de Linux kernel ontwikkeld heeft.

Wat is GitHub?

GitHub is a web-based hosting service for version control using Git. It is often referred to as simply " GitHub".

Laten we een terminal configureren.

Wat is je naam en je email ?

Eerst moeten we ingeven wie we zijn.

$ git config --global user.name "Jouw naam"

$ git config --global user.email "Jouw email"

git acties

$ git clone

Git clone is een comonthando die een online repository importeerd naar de map waarin je staat. Het is eigenlijk hetzelfde als een download van een repository.

Voorbeeld:

$ git clone ssh:repository.git
