Wat is Git?

Git is een versiebeheersysteem voor broncode.

Versiebeheersysteem? Wat is dat?

Een versiebeheersysteem is een programmaatje die je gebruikt om verschillende versies van uw code op te slaan.
Bijvoorbeeld de welbekende: "2.1","5.2","16.5" etc... zijn versies van software.

Dit wordt meestal gebruikt in combinatie met een website die uw code opslaat op het internet (bv: GitHub). Meer info over GitHub in het volgende deel.

Weetje: Git werd oorspronkelijk ontwikkeld door Linus Torvalds die initieel ook de Linux kernel ontwikkeld heeft.

Wat is GitHub?

GitHub is een plaats waar je online alle Git repositories kan opslaan. Voordeel daarvan is dat je online altijd je code kan terugvinden. M.a.w. wat een stick vroeger deed, kan je nu doen met GitHub.
Een ander voordeel is dat je hiermee kunt samenwerken met anderen. Iedereen kan code aannpassen en/of toevoegen aan je repo.

Laten we de terminal configureren.

Om Git en GitHub ten volste te kunnen gebruiken, moet Git ook eerst weten wie je bent.

Laten we de beginnen en de terminal open.

git config --global user.name "Jouw naam" git config --global user.email "Jouw email"

Laten we nu een GitHub account maken.

https://www.github.com

Nu kunnen we de echte cursus beginnen!

git acties

$ git clone

Git clone is een commando die een online repository importeerd naar de map waarin je staat. Het is eigenlijk hetzelfde als een download van een repository.

Voorbeeld: git clone ssh:repository.git

Extra GitHub pages

GitHub pages is een service die je kubt gebruiken om je website te publiceren op het internet. (let op, github pages is momenteel enkel beschikbaar voor statische websites)
